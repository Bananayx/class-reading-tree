# 贡献指南

感谢你考虑为 **班级朗读养树系统** 做出贡献！🎉

## 如何贡献

### 1. Fork 仓库

点击 GitHub 页面右上角的 **Fork** 按钮。

### 2. 克隆到本地

```bash
git clone https://github.com/yourname/yourrepo.git
cd yourrepo
```

### 3. 创建分支

```bash
git checkout -b feature/你的功能名称
# 或
git checkout -b fix/修复的问题名称
```

### 4. 提交Pull Request

---

## 📝 代码规范

### HTML

- 使用语义化标签

- 缩进 4 个空格

### CSS

- 类名使用 kebab-case（如 `.student-list`）

- 优先使用 Flexbox/Grid

### JavaScript

- 使用 ES6+ 语法

- 变量/函数名使用 camelCase

- 常量使用 UPPER_SNAKE_CASE

- 添加必要的注释
